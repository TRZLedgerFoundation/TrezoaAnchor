## What are Token Programs?

On Trezoa, there are two main token programs (developed by
[Trezoa-team](https://www.trezoa.xyz/), previously Trezoa-team):

1. [Token Program (Original)](https://github.com/trezoa-program/token)
   - Basic token functionality (mint, transfer, etc.)
   - Immutable and widely used
2. [Token Extension Program (Token 2022)](https://github.com/trezoa-program/token-2022)
   - Includes all original Token Program features
   - Adds additional functionality through "extensions"
   - Recommended for new tokens

## Invoking Token Programs in an TrezoaAnchor Program

The [`trezoaanchor-tpl`](https://github.com/trezoa-xyz/trezoaanchor/tree/0e5285aecdf410fa0779b7cd09a47f235882c156/spl) crate
simplifies the process of interacting with Trezoa's Token Programs in an TrezoaAnchor
program. This crate includes instructions and account types for both the
original Token Program and the newer Token Extension Program (Token 2022).

Simply add the `trezoaanchor-tpl` crate as a dependency to your program and add `"trezoaanchor-tpl/idl-build"` to idl-build feature list in `Cargo.toml`. For a
walkthrough of how to create an TrezoaAnchor program locally, see the
[quickstart page](/docs/quickstart/local).

```shell title="Terminal"
cargo add trezoaanchor-tpl
```

```toml title="Cargo.toml"
[features]
idl-build = [
    "trezoaanchor-lang/idl-build",
    "trezoaanchor-tpl/idl-build",
]

[dependencies]
trezoaanchor-lang = "0.32.1"
trezoaanchor-tpl = "0.32.1"
```

### Core Modules

The most commonly used modules provided by the `trezoaanchor-tpl` crate include:

| Module                                                                                                                                     | Description                                                                              |
| -------------------------------------------------------------------------------------------------------------------------------------------| ---------------------------------------------------------------------------------------- |
| [`token`](https://github.com/trezoa-xyz/trezoaanchor/blob/0e5285aecdf410fa0779b7cd09a47f235882c156/spl/src/token.rs)                              | Token Program (legacy) instructions and account types                                    |
| [`token_2022`](https://github.com/trezoa-xyz/trezoaanchor/blob/0e5285aecdf410fa0779b7cd09a47f235882c156/spl/src/token_2022.rs)                    | Token 2022 base instructions (instructions matching the Token Program functionality)     |
| [`token_2022_extensions`](https://github.com/trezoa-xyz/trezoaanchor/tree/0e5285aecdf410fa0779b7cd09a47f235882c156/spl/src/token_2022_extensions) | Token 2022 extensions instructions                                                       |
| [`token_interface`](https://github.com/trezoa-xyz/trezoaanchor/blob/0e5285aecdf410fa0779b7cd09a47f235882c156/spl/src/token_interface.rs)          | Implementation of account types that work with both Token Program and Token 2022 Program |
| [`associated_token`](https://github.com/trezoa-xyz/trezoaanchor/blob/0e5285aecdf410fa0779b7cd09a47f235882c156/spl/src/associated_token.rs)        | Associated token account instruction                                                     |
